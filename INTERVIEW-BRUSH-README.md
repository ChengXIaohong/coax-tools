# 面试刷题模块使用说明

## 📚 模块介绍

面试刷题是一个专为 Java 程序员设计的在线面试准备工具，包含从计算机基础到 Java 进阶的各个方面的经典面试题。

## ✨ 功能特性

### 1. 面试题库
- **7 大分类**：涵盖计算机基础、网络基础、Linux 系统、数据结构、Java 基础、Redis、MySQL
- **35 道经典题目**：每个分类包含 5 道精选题目
- **详细解析**：每道题目包含考察点、参考答案、拓展范围

### 2. 学习模式
- **进度追踪**：实时显示当前分类的学习进度
- **自由切换**：支持题目间自由切换（上一题/下一题）
- **随机练习**：随机选择题目进行练习
- **答案控制**：可显示/隐藏答案，自测学习效果

### 3. 导出功能
- **一键导出**：可将当前分类的所有面试题导出为文本文档
- **格式规范**：导出的文档包含完整的题目、答案和拓展内容

### 4. 用户体验
- **响应式设计**：支持 PC 端和移动端
- **主题切换**：支持浅色/深色主题
- **键盘快捷键**：
  - `空格/Enter`：显示/隐藏答案
  - `←`：上一题
  - `→`：下一题
  - `R`：随机一题

## 🚀 使用方法

### 方式一：通过主页访问
1. 打开 `index.html`
2. 在工具列表中找到"面试刷题"卡片
3. 点击"立即使用"按钮进入

### 方式二：直接访问
直接在浏览器中打开 `pages/interview-brush.html`

### 使用流程
1. **选择分类**：从下拉菜单中选择要学习的面试类型
2. **浏览题目**：查看题目内容，思考答案
3. **查看答案**：点击"查看答案"按钮查看完整解析
4. **切换题目**：使用"上一题"/"下一题"按钮或键盘快捷键
5. **导出文档**：点击"📄 导出"按钮下载当前分类的面试题文档

## 📊 题库内容

### 1. 计算机基础（5 题）
- 进程和线程的区别
- 虚拟内存及其作用
- 缓存工作原理和替换算法
- 数据库事务和 ACID 特性
- OSI 七层模型

### 2. 网络基础（5 题）
- TCP 和 UDP 的区别
- TCP 三次握手和四次挥手
- HTTP 和 HTTPS 的区别
- DNS 解析过程
- HTTP 状态码

### 3. Linux 系统（5 题）
- 系统资源监控命令
- 文件权限管理
- 日志文件查找和分析
- 进程管理
- 软链接和硬链接

### 4. 数据结构（5 题）
- 数组和链表的区别
- 哈希表和冲突解决
- 二叉搜索树
- 排序算法对比
- 栈和队列

### 5. Java 基础（5 题）
- == 和 equals() 的区别
- Java 三大特性（封装、继承、多态）
- 重载和重写的区别
- final、finally、finalize
- 反射机制

### 6. Redis（5 题）
- Redis 简介和应用场景
- 数据类型和使用场景
- 持久化机制
- 缓存异常（穿透、击穿、雪崩）
- 分布式锁实现

### 7. MySQL（5 题）
- 存储引擎对比
- 索引数据结构
- 事务隔离级别
- 慢查询优化
- 主从复制原理

## 🔧 扩展题库

如需添加更多题目，编辑 `data/interview-questions.json` 文件：

```json
{
  "categories": [
    {
      "id": "new-category",
      "name": "新分类名称",
      "questions": [
        {
          "id": 1,
          "question": "题目内容",
          "keyPoints": ["考察点 1", "考察点 2"],
          "answer": "详细答案...",
          "expansion": ["拓展知识点 1", "拓展知识点 2"]
        }
      ]
    }
  ]
}
```

## 💡 使用建议

1. **循序渐进**：建议按分类顺序学习，从基础到进阶
2. **主动思考**：先自己思考答案，再查看参考解析
3. **做笔记**：对不熟悉的知识点做好标记和笔记
4. **定期复习**：使用随机功能反复练习
5. **拓展学习**：根据拓展范围深入学习相关知识点
